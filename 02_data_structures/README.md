# Data Structures in Python

This folder contains my practice and understanding of core Python data structures
which are heavily used in Data Science and problem solving.

## Topics Covered
- Dictionary
- List
- Tuple
- Set

## Why Data Structures?
Data structures help in organizing, storing, and processing data efficiently.
In Data Science, they are commonly used for:
- Handling structured data
- Frequency counting
- Feature representation
- Data preprocessing

## Format
- Each topic is covered using a Jupyter Notebook
- Includes explanation, examples, and practice problems
